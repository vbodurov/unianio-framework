# Unianio Framework

**UNI**-ty

**ANI**-mation

**O**-rganizer

Unianio is unity framework for **procedural 3D humanoid character animations**

In addition to that, the framework provides:

 * Messaging
 * Inversion of Control
 * Scheduling
 * 3D Geometry and Math utilities

Download Unity package:
